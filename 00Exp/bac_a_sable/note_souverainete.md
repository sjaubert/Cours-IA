# Souveraineté des données

Une IA locale traite les données sur la machine de l'utilisateur. Aucun octet ne part vers un serveur externe.

Conséquences concrètes :

- Les secrets industriels ne quittent pas l'atelier.
- Le service fonctionne sans connexion internet (mode avion).
- Pas de dépendance à un fournisseur cloud qui peut changer ses prix ou fermer une API.

Limite : la qualité dépend du matériel local. Un petit modèle reste moins capable qu'un grand modèle cloud.
