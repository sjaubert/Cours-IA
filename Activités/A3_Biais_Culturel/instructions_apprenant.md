# Activité 3 : Le Biais Culturel et Linguistique

**Objectif :** Découvrir si l'IA a une culture "par défaut" et comment nos instructions peuvent l'orienter.

---

### Vos Instructions

Cette activité se déroule en deux temps.

#### Partie 1 : La Vision "Typique"

1.  **Lancer le script :** Double-cliquez sur le fichier `1_prompt_generique.bat`.
2.  **Analyser le texte :** Le fichier `sortie_generique.txt` va s'ouvrir.
3.  **En binôme, discutez :**
    * Quelle est la scène décrite ?
    * Quels sont les marqueurs de "réussite" ? (ex: promotion, argent, statut...)
    * Ces marqueurs vous semblent-ils vraiment "universels" ?

#### Partie 2 : La Vision "Orientée"

1.  **Lancer le script :** Double-cliquez sur le fichier `2_prompt_specifique.bat`.
2.  **Analyser le texte :** Le fichier `sortie_specifique.txt` va s'ouvrir.
3.  **En binôme, comparez les deux textes :**
    * Quelles sont les différences fondamentales de valeurs entre les deux scènes ?
    * Pourquoi l'IA n'a-t-elle pas proposé la seconde vision dès le début ?

Préparez-vous à partager vos conclusions lors du débriefing collectif.
