L'IA au Service de l'Entreprise

Connecter Gemini à vos données réelles : Google Workspace, Fichiers et Automatisation.

1. Les Fondations Techniques

Avant de transformer notre productivité, nous devons préparer notre environnement de travail. Voici les outils essentiels pour communiquer avec l'IA.

🛠️ La Boîte à Outils

Assurez-vous que les éléments suivants sont installés sur votre poste de travail :

  • Node.js (v20+) : Le moteur qui fera tourner nos scripts. Télécharger
  • Git & Git Bash : Pour récupérer les outils et exécuter les commandes Linux sous Windows. Télécharger
  • Go (Langage) : Nécessaire pour certaines extensions MCP. Télécharger
  • Google Cloud SDK : Pour l'authentification sécurisée. Installer
Important : Assurez-vous d'avoir installé l'extension Chrome "Google Antigravity" si vous utilisez l'interface web, ou préparez-vous à utiliser le terminal.

🔑 Configuration de la Clé API

Pour que Gemini fonctionne, il a besoin d'une clé d'accès (API Key). Une fois obtenue via Google AI Studio, nous devons l'inscrire durablement dans votre système Windows.

Ouvrez votre terminal (Invite de commandes ou PowerShell) et tapez :

>setx GEMINI_API_KEY "AIzaSy..." REM Remplacez par votre clé réelle

Note : Redémarrez votre terminal après cette commande pour qu'elle soit prise en compte.

Mise à jour de Gemini CLI vers la dernière version :

>npm install -g @google/gemini-cli@latest

2. Le Pont vers Google Workspace

Nous allons installer ToolsForMCPServer. C'est une extension révolutionnaire développée par Kanshi Tanaike qui permet à Gemini de "parler" directement à vos Google Sheets, Docs, et Scripts.

1Installation du CLI & Authentification

Si ce n'est pas déjà fait, installez le client Gemini et connectez-vous :

>npm install -g @google/gemini-cli
>gemini login

Cela ouvrira une fenêtre navigateur pour valider l'accès à votre compte Google.

2Copie du Projet "Cerveau" (Google Apps Script)

Nous devons déployer un petit script sur le cloud de Google qui servira d'intermédiaire.

  1. Connectez-vous à votre compte Google.
  2. Cliquez sur ce lien pour copier le projet source :
    Copier le Projet Script
  3. Le projet va s'ouvrir automatiquement dans l'éditeur de script.

3Déploiement de l'Application Web

C'est l'étape cruciale pour rendre le script accessible à votre IA locale.

  • Dans l'éditeur, cliquez en haut à droite sur Déployer > Nouveau déploiement.
  • Type de déploiement : cliquez sur la roue dentée et choisissez Application Web.
  • Description : "Serveur MCP Gemini".
  • Exécuter en tant que : Choisissez Moi (Votre compte).
  • Qui a accès : Choisissez Tout le monde (Anyone).
⚠️ Sécurité : Choisir "Tout le monde" permet à votre script d'être appelé depuis Internet, ce qui est nécessaire pour l'outil MCP local. La sécurité est assurée par l'URL unique (impossible à deviner) et la clé d'accès (accessKey) dans le code.

Cliquez sur Déployer, autorisez l'accès, puis copiez l'URL de l'application Web (elle finit par /exec).

4Configuration des Variables d'Environnement

Nous devons dire à votre ordinateur où se trouve ce script. Sous Windows, utilisez la commande suivante :

>setx MCP_WEB_APPS_URL "https://script.google.com/macros/s/###/exec?accessKey=sample"

Attention : Remplacez l'URL par la vôtre. Si vous n'avez pas changé le code du script, laissez accessKey=sample.

Redémarrez votre terminal une nouvelle fois.

5Installation de l'Extension

Lions le tout dans Gemini :

>gemini extensions install https://github.com/tanaikech/ToolsForMCPServer-extension

Pour tester, lancez gemini puis tapez /mcp. Si vous voyez une liste d'outils, la connexion est établie !

3. La Mémoire Documentaire

Pour que l'IA puisse travailler sur vos fichiers locaux (PDF, CSV, Rapports), nous installons l'extension FileSearchStore.

Installation Rapide

>gemini extensions install https://github.com/tanaikech/FileSearchStore-extension

Vérifiez l'installation avec la commande :

>/mcp desc

Vous devriez voir apparaître les 11 outils de gestion de fichiers.

4. Ateliers Pratiques : L'IA au Travail

Maintenant que la machine est prête, passons aux scénarios réels. Ces exercices sont conçus pour simuler des tâches de cadres en entreprise.

Environnement : Ces commandes sont à taper dans l'interface Gemini CLI (après avoir lancé la commande gemini).

Scénario : Stratégie & Synthèse

🧠 Le Synthétiseur de Connaissances

Contexte : Vous devez présenter demain matin les tendances du marché des "Semi-conducteurs". Vous avez téléchargé un livre blanc technique de 50 pages (PDF).

L'Action : Au lieu de lire, nous demandons à l'IA d'analyser, synthétiser et préparer vos slides.

Prompt (Commande) :

@file-search-store Ajoute le fichier 'market_report_2024.pdf' à ta mémoire.
Ensuite, analyse-le et fournis-moi :
1. Un résumé exécutif en 5 points clés.
2. Une liste des 3 risques majeurs cités.
3. Le plan détaillé d'une présentation PowerPoint de 10 slides basée sur ce contenu.
Scénario : Relation Client

📊 L'Analyste de Feedback Client

Contexte : Le service client vous a envoyé un export brut (CSV) des 500 derniers retours clients. La direction veut savoir "ce qui ne va pas".

L'Action : Analyse sémantique et proposition d'actions correctives.

Prompt (Commande) :

@file-search-store Lis le fichier 'retours_clients.csv'.
Agis comme un Directeur de l'Expérience Client.
1. Identifie les 3 thèmes de plainte les plus récurrents.
2. Pour chaque thème, cite un verbatim client "choc".
3. Rédige une proposition de réponse diplomate et empathique pour le problème numéro 1.
Scénario : Ressources Humaines

🤝 L'Assistant de Recrutement 2.0

Contexte : Vous avez reçu 50 CVs au format PDF dans un dossier pour un poste de "Chef de Projet Senior". Le tri manuel est fastidieux.

L'Action : Filtrage intelligent sur critères spécifiques.

Prompt (Commande) :

@file-search-store Scanne tous les fichiers PDF du dossier 'Candidats'.
Je cherche un profil spécifique. Crée un tableau comparatif (Markdown) des candidats qui répondent à CES deux critères :
1. Plus de 5 ans d'expérience en gestion de projet.
2. Mentionnent 'Agile' ou 'Scrum' et un niveau d'anglais courant.
Ajoute une colonne 'Score de pertinence /10' basé sur leur expérience.
Scénario : Administration & Automation

⚡ L'Orchestrateur Google Workspace

Contexte : Vous devez organiser une réunion de crise suite aux résultats financiers et envoyer les invitations.

L'Action : Interagir directement avec Gmail et Calendar via le serveur MCP.

Prompt (Commande) :

@mcp-server Utilise mes outils Workspace.
1. Cherche dans mes emails non lus celui provenant de 'finance@entreprise.com' contenant 'Rapport Trimestriel'.
2. Résume le contenu de cet email.
3. Crée un brouillon de réponse dans Gmail pour accuser réception et proposer une réunion lundi prochain à 10h pour en discuter.
