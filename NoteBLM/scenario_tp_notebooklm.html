TP NotebookLM

Prise en Main d'une Nouvelle Machine Industrielle avec l'IA

Durée: 3h Niveau: Intermédiaire Production & Maintenance

Contexte du TP

Votre Mission

Vous êtes Responsable Production ou Responsable Maintenance dans une entreprise industrielle. Votre direction vient d'investir dans une nouvelle presse hydraulique automatisée de 60 tonnes pour améliorer la productivité.

Cette machine complexe arrive demain sur site. Vous disposez de :

  • Manuel technique constructeur (350 pages, en anglais)
  • Protocoles de sécurité internes de l'entreprise
  • Rapports d'incidents sur des modèles similaires dans d'autres sites
  • Normes ISO applicables (sécurité machines)
Problème : Vous avez seulement 48h pour :
  • Comprendre le fonctionnement de la machine
  • Identifier les risques critiques
  • Préparer les plans de maintenance préventive
  • Former les opérateurs aux procédures essentielles

Solution : Vous allez utiliser NotebookLM pour centraliser, analyser et extraire rapidement les informations critiques de tous ces documents.

Objectifs Pédagogiques

À la fin de ce TP, vous serez capable de :

  1. Centraliser des documents techniques hétérogènes dans NotebookLM
  2. Interroger intelligemment une base documentaire complexe
  3. Extraire des checklists et procédures opérationnelles
  4. Identifier les risques critiques et moyens de prévention
  5. Produire des livrables professionnels (rapports, infographies, présentations)
  6. Gagner un temps considérable sur l'analyse documentaire industrielle

Compétences Développées

  • Technique : Maîtrise de NotebookLM, analyse de documentation technique
  • Métier : Gestion de risques, maintenance préventive, conformité réglementaire
  • Transversale : Synthèse d'information, prise de décision rapide, communication

Déroulement du TP

1

Étape 1 : Centralisation Documentaire (30 min)

Objectif

Créer une base de connaissances unifiée dans NotebookLM en important tous les documents pertinents.

Actions à réaliser

  1. Créer un nouveau Notebook nommé Presse Hydraulique 60T - Mise en Service 2026
  2. Importer les sources suivantes (voir section "Documents Fournis") :
    • Manuel technique (PDF)
    • Consignes de sécurité (Word/PDF)
    • Rapports d'incidents (PDF)
    • Extraits normes ISO 12100 et ISO 13849 (PDF)
  3. Vérifier que toutes les sources sont bien chargées (panneau Sources à gauche)
  4. Générer une synthèse automatique pour avoir une vue d'ensemble
Résultat attendu : Un notebook avec 4-6 sources importées et une synthèse générale affichée.

Questions de réflexion

  • Quels types de fichiers NotebookLM accepte-t-il ?
  • Comment vérifier que l'import s'est bien déroulé ?
2

Étape 2 : Simplification & Extraction (45 min)

Objectif

Utiliser NotebookLM pour transformer des centaines de pages en informations actionnables.

Missions

Mission 2.1 : Checklist de Maintenance Hebdomadaire

Prompt suggéré :

"En te basant uniquement sur le manuel technique et les consignes de sécurité, extrais les 10 points critiques de maintenance préventive hebdomadaire pour cette presse hydraulique. Présente-les sous forme de checklist opérationnelle avec : action, durée estimée, et points de vigilance."

Mission 2.2 : Procédure de Démarrage Simplifiée

Prompt suggéré :

"Crée une procédure de démarrage simplifiée en 8 étapes maximum pour un opérateur niveau BAC PRO. Inclus les contrôles visuels pré-démarrage et les points de sécurité obligatoires."

Mission 2.3 : Identification des Pièces d'Usure

Prompt suggéré :

"Liste toutes les pièces d'usure mentionnées dans le manuel avec leur fréquence de remplacement recommandée et leurs références constructeur. Présente sous forme de tableau."
Résultats attendus :
  • Checklist de maintenance (10 points)
  • Procédure de démarrage (8 étapes)
  • Tableau des pièces d'usure
Astuce : Utilisez le bouton "Créer un guide d'étude" dans le panneau Studio pour générer automatiquement des supports de formation !
3

Étape 3 : Analyse de Risques (45 min)

Objectif

Identifier et prévenir les risques critiques en croisant plusieurs sources documentaires.

Missions

Mission 3.1 : Identifier les 3 Risques Majeurs

Prompt suggéré :

"En croisant le manuel technique, les rapports d'incidents et les protocoles de sécurité, identifie les 3 risques majeurs pour cette presse hydraulique. Pour chaque risque, indique :
  • Nature du risque (écrasement, projection, etc.)
  • Gravité potentielle (arrêt, blessure, etc.)
  • Situations déclenchantes
  • Moyens de prévention recommandés
Cite précisément tes sources."

Mission 3.2 : Conformité Normes ISO

Prompt suggéré :

"Vérifie si la machine respecte les exigences de la norme ISO 12100 concernant :
  • Les protections contre l'accès aux zones dangereuses
  • Les dispositifs d'arrêt d'urgence
  • Les interverrouillages de sécurité
Liste les points conformes et les éventuelles lacunes."

Mission 3.3 : Plan de Prévention Intégré

Prompt suggéré :

"Génère un plan de prévention pour l'installation et la mise en service de cette machine, incluant :
  • EPI (Équipements de Protection Individuelle) obligatoires
  • Procédures de consignation/déconsignation
  • Formation minimale requise des opérateurs
  • Contrôles périodiques obligatoires
"
Résultats attendus :
  • Rapport des 3 risques majeurs avec citations
  • Checklist de conformité ISO
  • Plan de prévention opérationnel
4

Étape 4 : Production de Livrables (30 min)

Objectif

Créer des supports professionnels pour communiquer efficacement.

Livrables à Produire

  1. Présentation PowerPoint
    • Utilisez le panneau Studio "Créer des slides"
    • Thème : "Mise en Service Presse Hydraulique 60T - Points Critiques"
    • Cible : Direction et équipe terrain
  2. Infographie
    • Utilisez le panneau Studio "Créer une infographie"
    • Sujet : "Les 3 Risques Majeurs et leurs Préventions"
    • Style : Professionnel, orientation portrait
  3. Audio Overview (optionnel)
    • Générez un podcast IA résumant les points clés
    • Durée cible : 5-10 minutes
    • Écoutez-le et notez ce qui est pertinent
Résultats attendus : 3 supports professionnels prêts à être partagés avec votre équipe.
5

Étape 5 : Restitution & Débriefing (30 min)

Présentation en Groupe

Chaque groupe présente (5-7 minutes) :

  • Les 3 risques identifiés et leurs solutions
  • La checklist de maintenance créée
  • Les livrables produits (slide, infographie)
  • Le temps gagné vs méthode manuelle (estimation)

Questions de Débriefing

  1. Combien de temps aurait pris cette analyse sans NotebookLM ?
  2. Quelles ont été les limites rencontrées ?
  3. Les citations fournies étaient-elles toujours pertinentes ?
  4. Comment pourriez-vous utiliser NotebookLM dans votre travail quotidien ?
  5. Quels autres cas d'usage industriels voyez-vous ?
