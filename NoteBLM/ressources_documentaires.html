Comment utiliser ces ressources ?

Ces documents sont destinés à être importés dans NotebookLM pour réaliser le TP de prise en main d'une nouvelle machine industrielle. Vous pouvez :

  • Télécharger les manuels PDF directement
  • Copier les liens pour import direct dans NotebookLM (URL)
  • Adapter ces ressources à votre contexte industriel spécifique

Pack Complet pour Formateurs

Vous êtes formateur ? Voici comment préparer votre session :

  1. Téléchargez 2-3 manuels PDF (Enerpac, Compac)
  2. Imprimez les guides ISO depuis INRS
  3. Créez 2 rapports d'incidents fictifs (1 page chacun)
  4. Préparez un document "Consignes Sécurité Entreprise" (adapter template)
  5. Testez l'import dans NotebookLM avant la session

Checklist Formateur

Avant le TP, assurez-vous d'avoir :

  • Au moins 1 manuel technique complet (350+ pages)
  • 1 document sécurité/norme ISO (extrait 20-30 pages)
  • 2 rapports d'incidents (réels ou fictifs, 1-2 pages chacun)
  • 1 template maintenance ou checklist
  • Accès NotebookLM vérifié pour tous les stagiaires

Conseils Pratiques

Pour l'Import dans NotebookLM

  • Format PDF privilégié : Meilleure reconnaissance que Word/images
  • Nommage clair : "Manuel_Presse_HP70.pdf" plutôt que "Doc1.pdf"
  • Taille optimale : PDFs < 50 MB pour import rapide
  • URLs directes : Certains sites (INRS) permettent l'import direct par lien
  • Langue : Privilégiez documents français pour stagiaires francophones

Adaptation à Votre Contexte

Ce TP peut être adapté pour :

  • Robotique collaborative Manuels robots UR/KUKA + normes ISO 10218
  • Convoyeurs automatisés Documentation Interroll + normes EN 619/620
  • Installations électriques Normes NF C + schémas électriques
  • Systèmes pneumatiques Catalogues Festo/SMC + guides maintenance
