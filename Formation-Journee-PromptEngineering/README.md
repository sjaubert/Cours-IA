# Formation Prompt Engineering - Programme d'une Journée

## À Propos

Ce dossier contient le matériel pédagogique complet pour une formation d'une journée sur le **Prompt Engineering**, destinée à des salariés en entreprise.

**Objectif** : Fournir aux stagiaires une excellente méthodologie et des outils immédiatement utilisables dans leur quotidien professionnel.

---

## Structure du Programme

### Matinée (3h30)

#### Atelier 1 - Les 6 Piliers d'un Prompt Parfait (1h00)

- **[Version Interactive HTML](Atelier1_6Piliers.html)** (Recommandée pour les stagiaires)
- [Version Guide Formateur](Atelier1_6Piliers_Guide.md)

Découverte de l'anatomie d'un prompt efficace :

1. Rôle & Contexte
2. Objectif Précis
3. Tâche Claire
4. Contraintes
5. Format & Exemples
6. Étapes Critiques

**Activités** : Exercices d'identification, jeu en équipe, quiz de validation

---

#### Atelier 2 - Exercices Pratiques (1h00)

- **[Version Interactive HTML](Atelier2_Exercices_Pratiques.html)** (Recommandée pour les stagiaires)
- [Version Guide Formateur](Atelier2_Exercices_Pratiques.md)

5 exercices progressifs pour maîtriser la construction de prompts :

- Email simple (débutant)
- Procédure qualité (intermédiaire)
- Quiz de formation (intermédiaire+)
- Rapport de synthèse (avancé)
- Challenge personnel (application métier)

**Point fort** : Peer review entre participants

---

#### Atelier 3 - Techniques Avancées (1h30)

- **[Version Interactive HTML](Atelier3_Techniques_Avancees.html)** (Recommandée pour les stagiaires)
- [Version Guide Formateur](Atelier3_Techniques_Avancees.md)

Maîtrise de 4 techniques professionnelles :

- **Zero-Shot** : Questions directes simples
- **Few-Shot** : Apprentissage par exemples
- **Chain-of-Thought** : Raisonnement guidé
- **Prompt Chaining** : Décomposition de projets complexes

**Outils remis** : Tableau comparatif, arbre de décision, quiz de synthèse

---

### Après-midi (3h30)

#### Atelier 4 - Cas d'Usage Métier (1h30)

⚠️ **À développer**

6 scénarios métier identifiés :

- RH : Offres d'emploi
- Commercial : Propositions
- Support Client : Réponses types
- Formation : Contenus pédagogiques
- Qualité : Procédures et rapports
- Communication : Contenus réseaux sociaux

---

#### Atelier 5 - Approche Critique (1h00)

⚠️ **À développer**

- Hallucinations de l'IA
- Biais dans les réponses
- Vérification des informations
- Itération et amélioration

---

#### Bilan & Boîte à Outils (1h00)

⚠️ **À développer**

Remise des ressources finales et plan d'action personnel

---

## Contenu des Guides

Chaque guide contient :

- **Objectifs pédagogiques** précis
- **Déroulé minute par minute** avec timing strict
- **Énoncés d'exercices complets** avec contextes détaillés
- **Corrigés commentés** avec explications pédagogiques
- **Grilles d'évaluation** pour les participants
- **Notes pour le formateur** : gestion du temps, points d'attention, variantes possibles
- **Liste du matériel** nécessaire

---

## Approche Pédagogique

### 70% Pratique - 30% Théorie

**Plus de 13 exercices concrets** répartis sur les 3 premiers ateliers :

- Progression du simple au complexe
- Cas tirés de situations professionnelles réelles
- Feedback immédiat après chaque exercice

### Engagement & Interaction

- Jeux en équipe
- Peer review (échange entre participants)
- Quiz interactifs
- Challenge personnel sur besoin métier réel

### Différenciation

- Exercices bonus pour les participants rapides
- Variantes selon le niveau du groupe
- Adaptation possible selon le métier (RH, Qualité, Commercial, etc.)

## Utilisation

### Pour le Formateur

1. Lire attentivement chaque guide dans l'ordre
2. Préparer le matériel listé dans chaque atelier
3. S'assurer d'avoir un chronomètre visible
4. Adapter les exemples selon le public si nécessaire
5. Imprimer les templates et grilles pour les participants

### Pour l'Adaptation

Les guides sont conçus pour être modulaires :

- Chaque section est autonome
- Les exercices peuvent être remplacés par des cas métier spécifiques
- Le timing peut être ajusté selon les variantes proposées
