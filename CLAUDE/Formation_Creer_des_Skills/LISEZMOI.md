# Formation - Créer des compétences (skills) pour Claude

Module de formation d'une journée (7 h) destiné à des stagiaires d'entreprise
(groupe mixte : non-développeurs et profils techniques). Élaboré à partir du cours
officiel Anthropic « Claude Code Skills » (6 vidéos) et de son support écrit.

## Contenu du répertoire

| Fichier | Usage |
|---|---|
| `Support_formateur_Creer_des_Skills.docx` | Guide d'animation : fiche pédagogique, déroulé minuté, notes d'animation, points de vigilance, corrigés, quiz. Réservé au formateur. |
| `Livret_stagiaire_Creer_des_Skills.docx` / `.pdf` | Manuel remis aux participants : contenu des 6 modules, exemples, encadrés, exercices, fiche mémo, glossaire. |
| `Travaux_pratiques/Cahier_TP_Creer_des_Skills.docx` | Quatre TP guidés pas à pas + corrigés. |
| `Travaux_pratiques/exemples-skills/` | Compétences d'exemple prêtes à l'emploi pour les TP (dont une compétence volontairement défectueuse pour le TP de dépannage). |

## Les six modules

1. Que sont les compétences ?
2. Créer sa première compétence (TP 1)
3. Compétences avancées : métadonnées, descriptions, allowed-tools, divulgation progressive, scripts (TP 2)
4. Compétences et autres fonctionnalités (CLAUDE.md, sous-agents, hooks, MCP)
5. Partager les compétences : dépôt, plugins, entreprise ; sous-agents (TP 3)
6. Dépanner les compétences (TP 4)
