# Exemples de compétences (skills)

Ce dossier contient des compétences prêtes à l'emploi pour les travaux pratiques.

Chaque compétence est un dossier contenant un fichier `SKILL.md`.

| Dossier | Rôle | Notions illustrées |
|---|---|---|
| `pr-description/` | Rédige une description de pull request | skill minimale (name + description + instructions) |
| `compte-rendu-reunion/` | Formate un compte rendu de réunion | skill personnelle métier |
| `revue-document/` | Relit un document en lecture seule | allowed-tools, divulgation progressive (references/) |
| `charte-redaction/` | Applique la charte de rédaction | norme d'équipe / projet |
| `_a-depanner/ma-revue/` | Compétence volontairement défectueuse | support du TP de dépannage |

## Installation pour les TP

Compétence personnelle (disponible dans tous vos projets) :
- macOS / Linux : copier le dossier dans `~/.claude/skills/`
- Windows : copier le dossier dans `C:/Users/<votre-identifiant>/.claude/skills/`

Compétence de projet (partagée via le dépôt) :
- copier le dossier dans `.claude/skills/` à la racine du dépôt.

Redémarrer Claude Code après toute copie ou modification.
