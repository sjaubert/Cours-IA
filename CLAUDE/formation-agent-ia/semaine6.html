🔄 Le cycle OODA appliqué à l'amélioration d'un agent

OODA est une méthode de décision rapide (issue du monde militaire, adoptée par l'industrie) : Observer → Orienter → Décider → Agir. Appliquée à un agent IA, elle devient une boucle d'amélioration continue.

O

👁️ Observer

Analysez les réponses de votre agent sans jugement. Que s'est-il passé exactement ?

Ex : « Claude a donné un taux de TVA de 5,5 % sur un produit taxé à 20 %. »

O

🧭 Orienter

Identifiez la cause racine de l'erreur. Est-ce le prompt ? Le modèle ? L'outil ?

Ex : « Le prompt système ne précise pas que les taux de TVA doivent être vérifiés avec web_search. »

D

🎯 Décider

Choisissez une action corrective précise et testable.

Ex : « Ajouter dans le prompt : "Pour tout taux ou chiffre réglementaire, utilise web_search pour vérifier." »

A

⚡ Agir

Appliquez la correction, reposez exactement le même test, comparez les scores.

Ex : « Score exactitude : 1/3 → 3/3 après ajout de l'instruction web_search. »

⚠️ Catalogue des 6 erreurs les plus fréquentes

1. Hallucination de chiffre

Claude donne un taux, un montant ou une date sans le vérifier

✅ Correction : activer web_search + ajouter « [À VÉRIFIER] » dans le prompt

2. Réponse hors rôle

Claude répond à des questions hors de son domaine défini

✅ Correction : ajouter dans le prompt « Si la question dépasse ton domaine, dis-le explicitement »

3. Format inadapté

Réponse en prose alors qu'un tableau est attendu, ou inversement

✅ Correction : préciser dans le prompt le format par défaut attendu

4. Réponse trop longue

Claude développe plus que nécessaire, noie l'essentiel

✅ Correction : ajouter « Résumé en 5 lignes maximum, sauf demande contraire »

5. Perte de contexte

Claude « oublie » des informations données plus tôt dans la conversation

✅ Correction : vérifier que conversationHistory est bien transmis · Résumer le contexte en début de session longue

6. Excès de prudence

Claude refuse de répondre ou sur-nuance des questions simples

✅ Correction : clarifier dans le prompt les cas où il peut répondre sans réserve

J'ai documenté ce que fait l'agent (fiche de 1 page pour mes collègues)

🚀 Mise en service
J'ai testé l'agent avec 2 collègues qui ne connaissent pas Claude
J'ai un fichier de backup de mon prompt système
Je sais où mettre à jour le modèle si une nouvelle version de Claude est disponible
