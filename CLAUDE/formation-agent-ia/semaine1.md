# Semaine 1 — Découvrir les agents IA avec Claude

> **Étape 1 · Semaine 1 sur 6**
> ⏱ 2 h 30 · 🌿 Niveau A — Claude.ai uniquement · 👥 Profils mixtes (RH, compta, managers)

---

## 🎯 Ce que vous allez savoir faire à la fin de cette séance

- ✅ Utiliser Claude.ai pour rédiger un prompt efficace
- ✅ Comprendre la différence assistant / agent IA
- ✅ Donner un rôle à Claude pour obtenir de meilleures réponses
- ✅ Créer un artefact (document réutilisable)
- ✅ Identifier 2–3 cas d'usage concrets dans votre métier

---

## 🧩 Les concepts clés de cette semaine

| Terme | Définition |
|---|---|
| 🤖 **Assistant IA** | Répond à vos questions, une à la fois. Vous guidez chaque étape. |
| 🧠 **Agent IA** | Décompose lui-même un problème complexe en sous-tâches et les exécute en chaîne. |
| ✍️ **Prompt** | Le message que vous envoyez à Claude. Plus il est précis, meilleure est la réponse. |
| 📄 **Artefact** | Document généré par Claude que vous pouvez copier, exporter ou réutiliser directement. |
| 🎭 **Rôle système** | Instruction en début de conversation : « Tu es un juriste RH expert en droit français… » |
| 🌿 **Niveau A** | Tout se fait sur claude.ai, dans votre navigateur. Pas de code, pas d'installation. |

> **📌 À retenir absolument**
> Un prompt = **Contexte + Rôle + Tâche + Format attendu**
> Claude n'a pas accès à Internet par défaut — vérifiez toujours les informations récentes.
> Relisez toujours la réponse : Claude peut se tromper (*hallucination*).

## 🛠️ Activité 1 — Mon premier prompt avec un rôle

⏱ 45 min · 🌿 Niveau A · 👤 Individuel

Choisissez le prompt correspondant à votre profil, copiez-le dans une **nouvelle conversation Claude.ai**, puis lisez et améliorez la réponse.

### 🟢 Profil RH — Veille droit du travail

```
Tu es un juriste expert en droit du travail français, pédagogue et rigoureux.

Je suis responsable RH dans une PME industrielle de 80 salariés.

Tâche : explique-moi en moins de 300 mots ce que change la dernière
réforme sur les congés payés pendant arrêt maladie, et ce que je dois
concrètement vérifier dans nos contrats de travail.

Format : une explication simple, puis une liste de 5 points d'action.
```

### 🔵 Profil Comptabilité — Actualité fiscale

```
Tu es un expert-comptable pédagogue spécialisé dans les PME françaises.

Je suis responsable comptable d'une entreprise industrielle sous TVA
au régime réel normal.

Tâche : explique-moi les principales obligations de déclaration TVA
en début d'année (CA3, DEB, etc.) et les erreurs fréquentes à éviter.

Format : une liste claire avec les échéances clés et les pièges courants.
```

### 🟣 Profil Manager — Veille innovation

```
Tu es un consultant en innovation industrielle avec 15 ans d'expérience
dans le secteur manufacturier.

Je suis responsable d'un atelier de production de 30 personnes dans
une PME métallurgique.

Tâche : donne-moi 5 innovations concrètes et accessibles (budget < 20 000 €)
que des PME industrielles similaires ont adoptées ces 3 dernières années
pour améliorer la productivité ou la sécurité.

Format : 5 fiches courtes (nom de l'innovation, bénéfice principal,
ordre de grandeur du coût).
```

### 📋 Étapes à suivre

1. Ouvrez **claude.ai** dans votre navigateur et connectez-vous.
2. Cliquez sur **« Nouvelle conversation »**.
3. Copiez-collez le prompt correspondant à votre profil.
4. Lisez la réponse. Est-elle précise ? Utile ? Y a-t-il des informations douteuses ?
5. Reformulez une partie du prompt pour obtenir une réponse plus courte ou plus ciblée.
6. Notez : *quelle modification a le plus amélioré la réponse ?*

> 💡 **Astuce** : Essayez d'ajouter à la fin de votre prompt :
> *« Si tu n'es pas certain d'une information, dis-le explicitement. »*

## 🛠️ Activité 2 — Créer un artefact réutilisable

⏱ 40 min · 🌿 Niveau A · 👤 Individuel

Un **artefact** est un document structuré généré par Claude que vous pourrez utiliser directement au travail. Choisissez votre prompt selon votre profil.

### 🟢 Profil RH — Modèle d'entretien annuel

```
Tu es un responsable RH expérimenté.

Crée-moi une trame d'entretien annuel d'évaluation adaptée à un opérateur
de production dans une PME industrielle. L'entretien doit durer 45 minutes.

Structure : objectifs de l'année écoulée · bilan des compétences · points
de satisfaction · axes de progrès · objectifs pour l'année à venir ·
besoins en formation.

Format : un tableau Word-compatible avec une colonne « Questions du manager »
et une colonne « Notes ».
```

### 🔵 Profil Comptabilité — Modèle de relance client

```
Tu es un expert en gestion de trésorerie pour PME.

Rédige 3 modèles de mails de relance pour factures impayées :
- 1er mail : retard de 7 jours (ton courtois)
- 2e mail : retard de 30 jours (ton ferme mais professionnel)
- 3e mail : retard de 60 jours (ton formel, mention du recours possible)

Inclue des espaces [à compléter] pour le nom du client, le numéro de
facture et le montant.
```

### 🟣 Profil Manager — Compte-rendu de réunion type

```
Tu es un manager opérationnel dans l'industrie.

Crée-moi un modèle de compte-rendu de réunion hebdomadaire d'équipe,
adapté à un atelier de production de 15-30 personnes.

Sections : présents / absents · points de sécurité · avancement production ·
problèmes rencontrés · actions décidées (responsable + délai) ·
points divers · date prochaine réunion.

Format : tableau structuré, prêt à être copié dans Word.
```

### 📋 Étapes à suivre

1. Dans une **nouvelle conversation** Claude.ai, copiez-collez le prompt de votre profil.
2. Lisez l'artefact généré. Est-il directement utilisable dans votre contexte ?
3. Demandez à Claude de modifier un élément : *« Adapte ce modèle pour une entreprise de 200 personnes »*.
4. Copiez le résultat final et collez-le dans un fichier Word sur votre poste.
5. Notez : *combien de temps vous aurait pris la rédaction sans Claude ?*
