# Convention de redaction des procedures d'intervention

Quand tu rediges ou modifies une procedure d'intervention, applique systematiquement
la structure et les regles ci-dessous.

## Structure obligatoire (dans cet ordre)

1. **Pre-requis securite** : EPI requis, habilitations, zone a baliser.
2. **Consignation** : energies a consigner (electrique, pneumatique, hydraulique),
   reference du cadenas, verification d'absence de tension ou de pression.
3. **Mode operatoire** : etapes numerotees, une action par etape.
4. **Verification** : controles a effectuer avant remise en service.
5. **Remise en service** : ordre de deconsignation, essai a vide.

## Regles de fond

- Une etape = une action verifiable. Pas de "et" qui masque deux operations.
- Toute valeur chiffree porte son unite et sa tolerance (ex. 45 N.m +/- 2).
- Aucune reference machine inventee : si une donnee manque, ecris [A COMPLETER].
- Vocabulaire imperatif a l'infinitif : "Consigner", "Verifier", "Reposer".

## A ne jamais faire

- Omettre l'etape de consignation, meme pour une intervention courte.
- Fusionner verification et remise en service.
