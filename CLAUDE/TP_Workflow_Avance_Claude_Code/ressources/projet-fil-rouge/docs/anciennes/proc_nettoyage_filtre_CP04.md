Nettoyage filtre compresseur CP-04

Habilitation : aucune particuliere. Duree 30 min.
Consigner l'air comprime, verifier la mise a zero du manometre.
Porter des lunettes. Ouvrir le carter, sortir la cartouche filtrante.
Souffler la cartouche a l'air sec, remonter.
Refermer le carter, deconsigner, controler la pression de service a 7 bar.
