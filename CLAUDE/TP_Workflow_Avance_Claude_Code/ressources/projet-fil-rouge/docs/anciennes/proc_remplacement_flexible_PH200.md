PROCEDURE - Remplacement flexible hydraulique presse PH-200

Materiel : flexible HP ref FX-200-A, cle plate 24, bac de recuperation.
Avant de commencer, couper l'alimentation electrique et purger la pression du circuit.
Mettre les gants et les lunettes. Baliser la zone.
Devisser le raccord superieur puis le raccord inferieur, recuperer l'huile dans le bac.
Reposer le flexible neuf, serrer les raccords a 45 N.m +/- 2.
Verifier l'absence de fuite en remontant doucement la pression.
Remettre sous tension et faire un essai a vide.
