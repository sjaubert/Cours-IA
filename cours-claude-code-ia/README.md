# Formation IA Avancée — Maîtriser l'Écosystème Claude Code
## Guide du formateur & Sommaire général

> **Public cible :** Salariés de l'industrie — Administratifs, Ingénieurs, Informaticiens, RH, Formateurs  
> **Niveau :** Intermédiaire (notions de base supposées acquises)  
> **Durée estimée :** 2 jours (14h) — adaptable par module  
> **Format :** Formation présentielle ou distancielle

---

## À propos de cette formation

Cette formation s'appuie sur l'analyse de **15 dépôts GitHub** de référence dans l'écosystème Claude Code — l'outil d'IA agentique d'Anthropic qui a révolutionné la manière dont les professionnels automatisent leurs tâches complexes.

Au-delà du simple "assistant IA", Claude Code représente une nouvelle génération d'outils capables de lire des fichiers, exécuter des commandes, orchestrer des équipes de sous-agents et piloter des workflows multi-étapes de façon quasi-autonome. En un an, il a déjà touché **plus de 20 millions de commits** sur un million de dépôts.

Cette formation vous donnera les clés pour comprendre, évaluer et exploiter cette technologie dans votre contexte professionnel — quel que soit votre métier.

---

## Objectifs pédagogiques

À l'issue de cette formation, les stagiaires seront capables de :

- Comprendre ce qu'est un agent IA et en quoi il diffère d'un simple chatbot
- Identifier les concepts clés de l'écosystème Claude Code (skills, hooks, subagents, MCP…)
- Évaluer les 15 ressources GitHub de référence et savoir lesquelles correspondent à leurs besoins
- Appliquer des workflows structurés pour des tâches métier concrètes
- Adopter les meilleures pratiques pour travailler efficacement avec des agents IA
- Construire une feuille de route personnalisée selon leur profil professionnel

## Guide du formateur

### Prérequis participants
- Avoir déjà utilisé un outil IA conversationnel (ChatGPT, Claude, Copilot…)
- Notions de base sur ce qu'est un fichier, un dossier, une commande
- Curiosité et ouverture d'esprit vis-à-vis des nouvelles technologies

### Matériel nécessaire
- Ordinateurs avec accès Internet
- Accès à un compte Claude (claude.ai) ou Claude API
- Optionnel : accès à GitHub pour explorer les dépôts en direct

### Recommandations pédagogiques
- **Commencer par le module 4** (cas d'usage métier) si le groupe est non-technique — cela ancre la formation dans le concret avant d'aborder les concepts
- **Alterner théorie et démonstration** : chaque concept doit être illustré en direct
- **Adapter les exemples** au secteur d'activité du groupe
- **Encourager les questions** — cette technologie évolue vite, l'honnêteté sur les limites est une plus-value

### Découpage suggéré sur 2 jours

**Jour 1 (matin)**
- Accueil et tour de table (30 min)
- Module 1 — Introduction (1h30)
- Module 2 — Concepts clés (2h)

**Jour 1 (après-midi)**
- Module 3 — Les 15 ressources GitHub (3h)
- Démonstrations en direct

**Jour 2 (matin)**
- Module 4 — Cas d'usage métier (3h)
- Ateliers pratiques par groupe métier

**Jour 2 (après-midi)**
- Module 5 — Méthodologies (2h)
- Module 6 — Guide de démarrage + exercices (2h)
- Bilan et questions/réponses
