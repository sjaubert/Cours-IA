# Catalogue Synthétique des Compétences (Skills) de l'Agent

*Vu le nombre très important de compétences disponibles (plus de 300), ce document synthétise les familles principales. Les compétences exactes peuvent être appelées par leur nom technique.*

---

## 🛠️ 1. Ingénierie & Développement Logiciel

Ces compétences activent les meilleures pratiques, les architectures recommandées et les normes strictes pour les langages et frameworks spécifiques.

| Nom de la compétence | Objectif principal | Exemple de demande (Prompt) |
| :--- | :--- | :--- |
| **`python-pro`** / **`fastapi-pro`** | Code Python moderne (Typage, Asynchrone, Optimisation). | *"Écris le backend de mon app. Utilise `python-pro` pour un code robuste et performant."* |
| **`react-best-practices`** | Standards React, Next.js, prévention des re-renders. | *"Agis en tant que `react-best-practices` et revois ce composant UI."* |
| **`rust-pro`** / **`csharp-pro`** | Systèmes avancés, sécurité mémoire et architectures .NET. | *"Développe ce module en Rust en appliquant strictement `rust-pro`."* |
| **`clean-code`** / **`tdd-workflow`** | Application des standards industriels de qualité de code et pilotage par les tests. | *"Refactorise cet algorithme avec `clean-code`. Écris les tests d'abord (`tdd-workflow`)."* |

### 💡 Comment combiner tout cela ? (Le Méta-Prompt Idéal)

Pour des tâches complexes, n'hésitez pas à **empiler** mes rôles. Plus vous serez précis sur les "cerveaux" à allumer, plus la réponse sera excellente.

**👉 Exemple d'une requête combinée :**
> *"Je dois construire un outil de correction de QCM pour mes cours. Incarne **`software-architecture`** pour concevoir l'architecture globale, utilise **`python-pro`** pour coder le moteur de traitement, et active **`security-auditor`** pour être sûr qu'aucun étudiant ne puisse falsifier les résultats. Respecte aussi le **`tdd-workflow`** en fournissant les tests en premier."*
