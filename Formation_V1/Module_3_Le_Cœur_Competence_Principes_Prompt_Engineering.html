Le Cœur de la Compétence – Principes du Prompt Engineering

Pôle Formation UIMM-CVDL (S. JAUBERT)

Objectifs Pédagogiques

  • Définir le "Prompt Engineering" : L'art de formuler des requêtes précises pour obtenir des réponses pertinentes.
  • Maîtriser la structure RTCCF (les 5 éléments d'un prompt efficace).
  • Appliquer les techniques de base : Zero-shot et Few-shot prompting.

Contenu du Cours

1. L'art du Prompt : Devenir un bon "manager" d'IA

L'IA est un assistant incroyablement compétent mais sans initiative. Le "Prompt Engineering" est l'art de lui donner des instructions claires pour qu'il passe d'exécutant médiocre à collaborateur expert.

2. La structure d'un Prompt Efficace : RTCCF

Un prompt de qualité doit contenir ces 5 éléments :

Élément Description Exemple
R - Rôle Dire à l'IA "qui" elle est. Cela charge le contexte et le ton appropriés. "Tu es un ingénieur en cybersécurité..."
T - Tâche L'action précise que vous attendez. "...rédige une note d'information..."
C - Contexte Les informations de fond nécessaires pour exécuter la tâche. "...sur la nouvelle faille 'XYZ' qui affecte nos serveurs."
C - Contraintes Les limites à respecter (ton, longueur, exclusions). "...en moins de 150 mots, destinée à des managers non-techniques."
F - Format La forme de la sortie attendue (liste, tableau, JSON, etc.). "...sous forme de liste à puces."

3. Techniques de Base

  • Zero-Shot Prompting : La demande directe. Vous ne donnez aucun exemple, juste la tâche. (Ex: "Traduis ce texte en allemand.")
  • Few-Shot (ou One-Shot) Prompting : Vous donnez 1 à 3 exemples de ce que vous attendez. C'est la meilleure façon d'enseigner à l'IA un format de sortie précis. (Ex: "Voici un exemple de traduction : ... Maintenant, traduis ceci : ...").

Activité Pratique : "Le Prompt Inefficace" (45 min)

Objectif : Appliquer la structure RTCCF pour transformer un mauvais prompt en un prompt expert.

Étape 1 : Le Prompt Vague (5 min)

En binôme, soumettez ce prompt à une IA. Notez la réponse.

Fais-moi un e-mail pour les clients.

Étape 2 : Le Prompt Expert (30 min)

Maintenant, réécrivez ce prompt en utilisant la structure RTCCF. Contexte : Vous lancez une nouvelle gamme de pompes industrielles (Modèle X1000) plus économes en énergie (-20%). L'e-mail doit informer les clients existants.

# RÔLE
Tu es un directeur marketing B2B pour une entreprise d'équipement industriel. Ton ton est professionnel, fiable et orienté vers les bénéfices client.

# TÂCHE
Rédige un e-mail marketing pour annoncer le lancement de notre nouveau produit, la pompe industrielle X1000.

# CONTEXTE
Le X1000 remplace l'ancien modèle X900. Son avantage principal est une réduction de 20% de la consommation énergétique. Il est aussi plus silencieux. La cible est nos clients existants qui possèdent le X900.

# CONTRAINTES
- L'objet de l'e-mail doit être accrocheur.
- Le texte ne doit pas dépasser 150 mots.
- N'utilise pas de jargon trop technique.
- L'appel à l'action est de "planifier une démonstration".

# FORMAT
Rédige l'objet de l'e-mail et le corps de l'e-mail.

Étape 3 : Comparaison (10 min)

Soumettez le nouveau prompt. Comparez la réponse 1 (vague) et la réponse 2 (expert). Discutez des différences en grand groupe.
