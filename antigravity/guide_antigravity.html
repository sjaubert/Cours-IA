Guide de Démarrage Antigravity

Antigravity est une plate-forme de développement "agent-first" qui intègre une IA autonome capable de planifier, coder et exécuter des tâches complexes pour accélérer vos projets.

1. Prérequis Indispensables

Avant de commencer, assurez-vous impérativement de disposer des éléments suivants sur votre poste :

  • Git : Le système de gestion de versions doit être installé.
    Télécharger Git
  • Python : Le langage Python doit être installé (cochez "Add Python to PATH" à l'installation).
    Télécharger Python
  • Compte Google : Un compte personnel est requis pour l'authentification.

2. Installation

  1. Téléchargez l'installateur pour votre OS sur le site officiel.
  2. Lancez l'installation et suivez les étapes.
  3. Au premier lancement, connectez-vous avec votre compte Google via le navigateur.
Télécharger Antigravity

3. Connexion à GitHub

Pour sauvegarder vos projets et collaborer, connectez Antigravity à votre compte GitHub :

  1. Ouvrez Antigravity.
  2. Accédez aux Paramètres (Settings) via l'icône d'engrenage.
  3. Allez dans la section Integrations ou Accounts.
  4. Sélectionnez GitHub et cliquez sur Connect.
  5. Autorisez l'accès dans la fenêtre de navigateur qui s'ouvre.

4. Utilisation de base

L'Agent

L'agent est votre copilote. Vous pouvez lui donner des instructions en langage naturel (ex: "Crée une page web pour..."). Il peut :

  • Planifier les étapes d'un projet.
  • Écrire et modifier du code.
  • Exécuter des commandes dans le terminal.

Modes de fonctionnement

  • Auto : L'agent prend des décisions autonomes (recommandé).
  • Terminal : Vous pouvez définir si l'agent a le droit d'exécuter des commandes seul (Auto) ou s'il doit demander la permission.
Sécurité : Pensez à configurer la liste des sites web autorisés dans les paramètres pour sécuriser la navigation de l'agent.
