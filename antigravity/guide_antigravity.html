Google Antigravity

Plate-forme de développement agentique

Qu'est-ce qu'Antigravity ?

Antigravity n'est pas un simple éditeur de code, mais une véritable plate-forme "agent-first" qui transforme l'IDE traditionnel en un centre de contrôle pour agents autonomes.

À la différence des assistants de codage standards qui se contentent d'autocompléter des lignes, Antigravity fournit un agent autonome capable de planifier, coder, et même naviguer sur le Web pour vous aider à créer vos projets.

Prérequis

Important : Antigravity est actuellement disponible en aperçu pour les comptes Gmail personnels avec un quota sans frais pour les modèles de premier plan.
  • Système d'exploitation : Mac, Windows ou certaines distributions Linux
  • Navigateur : Google Chrome
  • Compte : Compte Gmail personnel (obligatoire)
  • Niveau : Tous niveaux, y compris débutants

Installation

1Télécharger Antigravity

Rendez-vous sur la page Téléchargements et cliquez sur la version correspondant à votre système d'exploitation.

2Lancer l'installation

Exécutez le programme d'installation et suivez les étapes. Vous devrez configurer :

  • Thème de l'éditeur (clair ou sombre)
  • Mode de développement (voir ci-dessous)
  • Préférences de l'éditeur

3Se connecter avec Google

Connectez-vous avec votre compte Gmail personnel. L'authentification s'effectue via le navigateur Chrome.

Modes de Développement

Antigravity propose différents niveaux d'autonomie pour l'agent. Deux paramètres clés :

Règle d'exécution du terminal

  • Off : N'exécute jamais automatiquement les commandes
  • Auto : L'agent décide (recommandé pour débuter)
  • Turbo : Exécute toujours automatiquement

Règles relatives aux avis

  • Always Proceed : Pas de demande d'avis
  • Agent Decides : L'agent décide quand demander (recommandé)
  • Request Review : Demande toujours un avis
Recommandation : Choisissez le mode "Développement assisté par l'agent" qui offre un bon équilibre et permet à l'agent de prendre des décisions tout en revenant vers vous pour approbation.

Architecture d'Antigravity

Éditeur

Zone de code traditionnelle avec autocomplétion intelligente

Gestionnaire d'Agents

Centre de contrôle pour piloter vos agents autonomes

Navigateur Antigravity

L'agent peut naviguer sur le Web pour vous

Terminal

Exécution des commandes avec contrôle de sécurité

Raccourcis clés

  • Cmd + E : Basculer entre Éditeur et Gestionnaire d'agents
  • Cmd + L : Activer/désactiver le panneau de l'agent
  • Ctrl + ` : Activer/désactiver le terminal

Personnalisation

Règles

Les règles guident le comportement de l'agent (style de code, documentation, etc.). Ce sont comme des instructions système.

Exemple : "Assure-toi que tout le code respecte le guide de style PEP 8"

Workflows

Les workflows sont des requêtes enregistrées que vous pouvez déclencher à la demande avec /

Ils peuvent être globaux ou spécifiques à un espace de travail.

Sécurité

Sécurité du navigateur : Configurez une liste d'autorisation d'URL pour éviter que l'agent ne visite des sites malveillants ou compromis.

Accédez à Antigravity → Settings → Advanced Settings pour configurer la liste d'autorisation des URL de navigateur.

Cas d'Utilisation

Antigravity peut vous aider dans de nombreuses tâches :

  • Extraction d'informations depuis des sites web
  • Développement de sites web complets avec Python/Flask
  • Création d'applications de productivité
  • Génération de tests unitaires et validation automatique
  • Analyse et synthèse de données
